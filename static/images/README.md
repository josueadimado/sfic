# Image Placement Guide

Put your final images in this folder with these exact names:

- `hero-main.jpg` - main hero background
- `expectation.jpg` - image beside "What To Expect"
- `banner.jpg` - wide background for the mid-page banner
- `media-1.jpg` - gallery image 1
- `media-2.jpg` - gallery image 2
- `media-3.jpg` - gallery image 3
- `media-4.jpg` - gallery image 4
- `presenters.jpg` - image for the presenters split section

Recommended sizes:

- Hero: 1920x1080 or larger
- Wide banner: 1920x700
- Side/split images: 1200x900
- Gallery images: 900x700 each

Notes:

- Use `.jpg` for photos for faster loading.
- Keep file names exactly the same so the site picks them up automatically.
